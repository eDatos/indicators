========================================================================
Indicators -- Gestión de Sistemas de Indicadores del ISTAC
========================================================================

El Gestor de Sistemas de indicadores del ISTAC es una herramienta que
permite llevar a cabo la creación y gestión de indicadores y sistemas
de indicadores que posteriormente son visualizados en el portal web del
ISTAC. Además, este proyecto cuenta con los siguientes canales de difusión
pública:
- Portal del ISTAC
- Widgets
- API

Por favor, consulta nuestro gestor de incidencia para obtener más información
de las incidencias con el id [INDISTAC-xxx].
http://jira.arte-consultores.com/browse/INDISTAC
